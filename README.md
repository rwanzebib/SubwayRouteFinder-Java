# SubwayRouteFinder-Java
A simple Java project that finds direct subway routes between two NYC stations.

### Purpose
This project simulates a small version of Google Maps or the MTA app.  
You enter a **start station** and an **end station**, and the program tells you:

- Which trains stop at both stations  
- Whether there is a direct route  
- If you need a transfer  

### Concepts Used
- Java Maps ('HashMap')
- Arrays & Lists
- Loops
- Conditionals
- Input with `Scanner`
- Simple lookup algorithm
